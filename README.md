# Career Resource Guide: Emotional Intelligence

## Publish with GitHub Pages
1. Create a new **public** GitHub repository.
2. Upload `index.html` and `README.md`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose **main** and **/(root)**, then click **Save**.
6. Wait 1–3 minutes. GitHub will display your live website address.
7. Use that live address to create your real QR code.

The live address will usually look like:
`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

Test the site in a private browser window before creating the QR code.
